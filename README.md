# The Matrix Todo

A simple todo app for The Matrix fans. Keep track of your tasks while you try to defeat the machines!

https://the-matrix-todo.netlify.app/

## Features

- Mobile responsive
- Fullscreen mode
- Error handling
- Data persistence
- Hosted on Netlify

## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Rails API
